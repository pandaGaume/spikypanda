# Convolutional Neural Networks (CNN)

Convolutional Neural Networks (CNN) are a specialized type of ANN designed to process data with a grid-like topology, such as images.

They use convolutional layers that apply filters to detect local patterns like edges, textures, and shapes. This makes CNNs very effective for image classification, object detection, and other vision tasks.

CNNs typically consist of convolutional layers, pooling layers, and fully connected layers.
