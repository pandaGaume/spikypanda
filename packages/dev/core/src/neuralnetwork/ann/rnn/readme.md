# Recurrent Neural Networks (RNN)

Recurrent Neural Networks (RNN) are designed to handle sequential data by maintaining a memory of previous inputs through hidden states.

They are ideal for tasks where the order of input matters, such as time series prediction, natural language processing, and audio analysis.

Variants like LSTM (Long Short-Term Memory) and GRU (Gated Recurrent Unit) address the limitations of basic RNNs by managing long-term dependencies more effectively.
