# Multi-Layer Perceptrons (MLP)

A Multi-Layer Perceptron (MLP) is a class of feedforward artificial neural network (ANN). It consists of at least three layers of nodes: an input layer, one or more hidden layers, and an output layer.

Each neuron in one layer is fully connected to the neurons in the next layer. MLPs use non-linear activation functions (like ReLU, Sigmoid, or Tanh) to model complex relationships in data.

MLPs are suitable for:

- Classification of structured/tabular data
- Regression tasks
- Function approximation

While not as specialized as CNNs or RNNs, MLPs remain a fundamental building block in neural network design.
